# Classroom_Scheduler-2022

- Classroom-Scheduler is a free blended learning platform for educational institutions that aims to simplify creating timetables and distributing assignments.
- It is a web based application where Class-Coordinator is responsible for creating timetables adding teachers and students. 
- The primary purpose of Classroom-Scheduler is to streamline the process of sharing timetables and assignments between teachers and students Classroom Scheduler helps in centralizing data of students and teachers.
